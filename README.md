# ShiftNote · Epic Edition

A standalone, offline-first clinical shift documentation tool for nurses. No login. No cloud. No dependencies.

## 🔗 Live App
**[Open ShiftNote →](https://regelus19.github.io/shiftnote)**

## Features
- 🎙 **LOG NOW** — speak or type freely, auto-timestamped, auto-classified
- 🧠 **Epic Brain** — photograph your Epic Brain screen as a shift reference
- ✅ **Task Board** — pending task tracker with priority levels
- 📋 **Epic Note Generator** — Nursing Note, Shift Assessment, SBAR Handoff, and more
- 📱 **Install as app** — Add to Home Screen on iOS/Android, works fully offline
- 💾 **Auto-saves locally** — data persists if screen locks

## How to Use
1. Open the link above in Safari (iOS) or Chrome (Android)
2. Tap **Share → Add to Home Screen** to install
3. Configure your shift info and start logging

## Clinical Categories Auto-Detected
Assessment · Neuro · Cardiac/Hemodynamics · Respiratory · GI/GU · Medication Event · Procedure · Lines/Tubes/Drains · Critical Lab/Value · Provider Notification · Patient Education · Pain/Comfort · Fall/Safety · Transfer/Handoff

## Privacy
All data stays on your device. Nothing is transmitted anywhere. No accounts required.
